Ideas

I tried many bootstrap wysiwyg editors. They suck! We surely made a better one. Alan's code is much more efficient than others.

We made simple implementations.

  • Easy file upload with common server side languages
  • Auto-resize image file
  • Great table features and easy editing shortcuts
  • Providing nowadays interaction for text editing tool

We spent a less time for summernote in a own company but code grows up. and now some nice guys help us for debug, quality, idea
